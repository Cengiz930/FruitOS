🍎 Introducing FruitOS — The Minimalist Python-Powered Operating System 🍏

Welcome to FruitOS, a sleek and lightweight operating system crafted entirely in Python! 🚀 Designed with simplicity and efficiency as its core principles, FruitOS offers a clean, minimal environment perfect for learning, development, and experimentation. 🧑‍💻✨

🌟 Key Features of FruitOS:

  • 🐍 Python-Based: Built from the ground up using Python.
  • ⚡ Lightweight & Fast
  • 🎯 Focus on Essentials
  • 🛠️ Ideal for Education
  • 🔧 Customizable
  • 🌐 User-Friendly Interface

🖥️ FruitOS 1.0 – System Requirements

Minimum:
  • CPU: 8086/8088 compatible processor
  • RAM: 256 KB
  • Storage: 7 MB free disk space
  • Display: CGA or compatible graphics (4 colors)
  • Input: Keyboard
Recommended:
  • CPU: Intel 80286 or better
  • RAM: 640 KB
  • Storage: 10 MB free disk space
  • Display: VGA graphics (256 colors)
  • Input: Keyboard and mouse

🍇 FruitOS 2.0 — What’s New?

  • time → Displays current time
  • calc → Mini calculator
  • notepad → Write and display notes
  • shutdown → Simulates shutdown
  • version → Displays version info

🖥️ FruitOS 2.0 – System Requirements

Minimum:
  • Processor: Intel 80486 or compatible
  • Memory: 16 MB RAM
  • Storage: 15 MB disk space
  • Display: CGA, 320x200 or higher
  • Input: Keyboard
Recommended:
  • Processor: Intel Pentium (90 MHz or higher)
  • Memory: 32 MB RAM or more
  • Storage: 20 MB disk space
  • Display: VGA-compatible (640x480 or higher)
  • Input: Keyboard and mouse (optional)

🍉 FruitOS 3.0 — What's New?

  • Alarm system: alarm HH:MM sets alarms
  • Notepad app: write notes to note.txt
  • Calculator: calc evaluates math
  • Prompt customization: setprompt <new_prompt>
  • File system commands: ls, cd, mkdir, rm
  • User management: useradd, login, logout
  • Run external scripts: runscript <file>
  • Networking: ping, connect
  • shutdown / exit: clean exit
  • version / about: info commands
  • time: shows date & time
  • Multi-threaded alarm checker

🌟 Frutech AI - Your Friendly Python Chatbot

Frutech AI is a simple yet powerful chatbot built with Python. Designed for fun and easy interactions, Frutech AI can understand basic English inputs and respond with friendly messages.

  • 🗨️ Responds to basic greetings, questions, and farewells
  • ⚡ Runs fast with no external dependencies
  • 🔑 Keyword-based logic that’s easy to customize
  • 🤖 A great starting point for beginner AI projects
  • 🚀 Lightweight and ready to extend with advanced AI features
Note: Some antivirus software may flag new software like FruitOS or Frutech AI as suspicious (false positive). This does not mean the software is harmful.

How to Temporarily Disable and Re-enable Antivirus

Windows Defender

  • Open Windows Security → Virus & threat protection → Manage settings → Turn off real-time protection

Avast

  • Right-click system tray icon → Avast shields control → Disable

AVG

  • Right-click system tray icon → Disable protection

Norton

  • Open Norton → Settings → Turn off Auto-Protect

McAfee

  • Open McAfee → Real-Time Scanning → Turn off

Bitdefender

  • Open dashboard → Protection → Disable shield
